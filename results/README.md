| File | Contents |
|------|----------|
| `validation.npz` | `ns`, `theory`, `measured` (deconvolution penalty), `snrc_check` (= 43.16). |
| `clouds.npz` | `ssim/nrmse/pearson[method, fc, seed]` for methods (gls, white, b2); `fcs`, realized `covers`; reconstruction maps for seed 11 (`map_<m>_fc<i>`); static-Wiener image `b1img_fc3`; scene/raster/opacity snapshots. |

| `parts/` | Per-configuration checkpoints written by individual runs. |

Synthetic-input data dictionary:
- **Truth albedo map** — seeded (7) multi-octave Gaussian fields, 144×72
  lon-lat grid; ocean 0.06, land 0.32±0.10, polar caps ≤0.60, land
  fraction 0.30.
- **Cloud field** — advected Ornstein–Uhlenbeck opacity screen;
  correlation length 12°, advection 6°/day, τ_c = 4 d, cloud albedo
  0.65; mean cover set by a fixed calibrated threshold.
