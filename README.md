# sgl-time-domain-imaging

**Time-Domain Imaging of Rotating, Cloud-Covered Exoplanets with the
Solar Gravitational Lens: Reconstruction Algorithms, Observing-Cadence
Requirements, and Mission Targets**

Sanskar Sontakke (Independent Researcher)
[ORCID 0009-0008-4531-5707](https://orcid.org/0009-0008-4531-5707) · sanskarsontakke@gmail.com

This repository contains the complete paper, simulation code, and data
products. Every number, figure, and table in the manuscript regenerates
from the code in `src/` (fixed seeds, ~40 core-minutes total).

## Paper

📄 [`paper/sgl_time_domain_imaging.pdf`](paper/sgl_time_domain_imaging.pdf) — 12 pages, 7 figures, 2 tables.

**Headline results**

| Finding | Value |
|---|---|
| Rotating cloud-free planet, time-domain GLS | Pearson *r* = 0.99, SSIM 0.92 (prior art: SNR ≲ 2) |
| Cloud noise vs. coronal shot noise per sample | 14–23× larger (covers 22–72%) |
| Fidelity at Earth-like (72%) cloud cover | *r* ≈ 0.26 at fixed 90-d / 16-craft / 8 h-per-pixel budget |
| Observing-cadence law (fixed photons & duration) | 64 short revisits beat 4 long dwells: *r* 0.15 → 0.58 |
| Required a-priori rotation-period knowledge | ~10⁻⁴ (10⁻³ is fatal) |
| Best confirmed targets | Proxima b · Ross 128 b · GJ 1061 d · Teegarden c · GJ 273 b |

## Quickstart

```bash
pip install -r requirements.txt
cd src
python3 run_experiments.py setup          # caches operators (~10 s)
python3 run_experiments.py validation     # reproduces published benchmarks
python3 make_figures.py                   # figures from shipped results/
```

Full regeneration of all experiments:

```bash
cd src
python3 run_experiments.py setup
python3 run_experiments.py validation
for i in 0 1 2 3 4; do for j in 0 1 2; do python3 run_experiments.py clouds $i $j; done; done
for i in 0 1 2 3 4; do for j in 0 1; do
  python3 run_experiments.py cadence $i $j ds
  python3 run_experiments.py cadence $i $j solve
done; done
python3 run_experiments.py robust_tau
python3 run_experiments.py robust_sig
python3 run_experiments.py robust_prot 0
python3 run_experiments.py robust_prot 1
python3 run_experiments.py merge
python3 targets.py
python3 make_figures.py
```

Seeds: truth map 7 · campaign 2026 · realizations 11–13 ·
held-out calibration 99/199 · cloud-climatology probes 999.

## Physical conventions

Fiducial case follows the published SGL literature: exo-Earth at
z₀ = 30 pc observed from z = 650 AU, d = 1 m, λ = 1 μm;
Q_exo = 8.01×10⁴ s⁻¹, Q_cor = 6.20×10⁹ s⁻¹ (Turyshev & Toth 2022,
MNRAS 515, 6122); aperture-averaged kernel K(ρ) = d/4ρ normalized to
unit mean disk signal (Turyshev 2026, arXiv:2606.14899). The code
reproduces SNR_C = 43.16 per 1800 s dwell and the 0.891·D/(d√N)
deconvolution penalty as validation gates.

No real observational data are used; all inputs are seeded synthetic
scenes plus published constants.
